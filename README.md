# Java Chessboard

This was just a chessboard. I've updated it and now you can move pieces! However, there is
no move validation yet (you can move the rooks diagonally!).

## Compilation and running
```
cd src
javac *.java -d ..
cd ..
java App
```
Tested on Ubuntu 20.04 64-bit
